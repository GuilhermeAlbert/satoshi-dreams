# Contributing

Thanks for your interest in contributing to Satoshi Dreams.

This project is intentionally simple and community-friendly.

## Ways to contribute

- Add new items
- Improve copywriting
- Translate content
- Improve accessibility
- Improve visual consistency
- Suggest funny or iconic ideas
- Improve performance

## Adding a new item

Every item should include:

- name
- category
- short description
- approximate BTC value
- illustration

Keep descriptions playful, short, and elegant.

## Design direction

We prefer:

- warm colors
- editorial illustrations
- clean layouts
- calm typography
- premium minimalism

Avoid:

- aggressive meme aesthetics
- low quality AI spam
- cluttered layouts

## Pull Requests

Keep pull requests focused and easy to review.

Small contributions are encouraged.
